# Registration Demo – Full Stack Application

A full-stack registration and authentication demo built with a clear separation between frontend and backend, following real-world project structure and best practices.

## Project Structure

registration-demo/
├── frontend/
├── backend/
└── README.md

## Overview

This repository demonstrates how modern applications handle user authentication using a clean and scalable architecture.

## Tech Stack

Frontend:

- Modern JavaScript Framework
- REST API integration
- Token-based authentication

Backend:

- Node.js
- Express.js
- Mysql (Sequelize)
- JWT Authentication

## How to Run

Backend:
cd backend
npm install
npm run dev

Frontend:
cd frontend
npm install
npm run dev
